# Ovládání zoomu rukou
Malý projekt v Pythonu, kde můžeš ovládat rukou ZOOM webkamery pomocí pohybu prsty. Stojí to na knihovnách OpenCV a MediaPipe.

# Co k tomu potřebuješ
Než to pustíš, musíš mít nainstalované tyhle dvě věci:

pip install opencv-python mediapipe

Jak to spustit
1. Stačí mít připojenou webkameru.
Spusť to přes terminál:
python HandTracking.py

# Jak se s tím pracuje
Zoom: Stačí dát palec a ukazováček k sobě nebo od sebe – obraz se podle toho plynule přiblíží nebo oddálí.
Konec: Když chceš skončit, ukaž na kameru obě dlaně. Vyskočí menu, kde stačí potvrdit ukončení palcem nahoru.
Kdykoliv můžeš aplikaci vypnout klávesou q na klávesnici.
